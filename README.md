# 💖 Portal do Afeto

Bem-vindo(a) ao código-fonte do nosso amor. Este é um projeto desenvolvido para materializar em formato digital um pouco de tudo o que sinto por você.

> "A tecnologia nos conecta com o mundo, mas este portal me conecta diretamente com você."

## 🌌 O que é este projeto?

O **Portal do Afeto** é um ambiente interativo, desenhado com estética cinematográfica, onde cada seção é um pouco do meu amor por você. A interface é fluida, amigável e cheia de pequenas surpresas preparadas com carinho.

Aqui você vai encontrar:
- **Primeiro Encontro**: Um lembrete visual de onde tudo começou, com um coração que transborda a cada toque na tela.
- **Chuva de Corações**: Um portal interativo para criar uma chuva de corações na tela, permitindo personalizar o estilo e a intensidade de cada um.
- **Galeria de Elogios**: O espaço reservado para as palavras mais bonitas que tenho para você, exibidas em uma galeria suave e infinita.
- **Mil Te Amos**: Mais de 100 formas de dizer o quanto você é especial, organizadas por proximidade cultural e acompanhadas de um guia de pronúncia.
- **Pedido Especial**: O portal final, guardado com todo o cuidado para uma pergunta muito importante

## ✨ Diferenciais Técnicos

- **Interatividade Global**: Toque em qualquer lugar da tela e veja um coração nascer com a cor temática da página.
- **Identidade Visual Dinâmica**: Cada seção possui sua própria paleta de cores (Violeta, Rosa, Ciano, Laranja e Magenta) que se adapta automaticamente para criar o clima perfeito.
- **Performance Otimizada**: Sistema inteligente de gestão de memória que remove elementos animados do navegador assim que eles cumprem sua missão, mantendo o portal leve e fluido.

## 🛠️ Tecnologias Envolvidas

- `HTML5` (A estrutura do nosso relacionamento)
- `Vanilla CSS / Animações` (As cores e o brilho dos nossos dias, com efeitos de Vidro e Mesh Gradients)
- `Modern Layouts (Grid & Flexbox)` (A organização perfeita para o nosso Bento Grid responsivo)
- `CSS Variables` (A identidade visual que muda de cor conforme o clima de cada página)
- `JavaScript` (A lógica e a dinâmica que fazem o nosso amor funcionar)
- `SVG & ClipPath` (A técnica fluida que faz o coração encher com perfeição)
- **100% Amor** (O framework principal utilizado em todo o processo)
