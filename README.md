O **Portal do Afeto** é um ambiente interativo, desenhado com estética cinematográfica, onde cada seção reflete uma pouco do meu amor por você. A interface é fluida, amigável e cheia de pequenas surpresas.

## ✨ Funcionalidades Principais

- **Primeiro Encontro**: Um lembrete visual de onde tudo começou, com um coração que transborda a cada toque.
- **Chuva de Corações**: Interação em tela cheia para criar uma chuva de corações, permitindo curtir a magia do momento.
- **Galeria de Elogios**: O espaço reservado para as palavras mais bonitas que tenho para você, exibidas em uma galeria suave e infinita.
- **Mil Te Amos**: Mais de 100 formas em diferentes idiomas de dizer o quanto você é especial, organizadas poeticamente.
- **Pedido Especial**: O portal final, guardado com todo o cuidado para uma pergunta muito importante. 💍
- **Interatividade Global**: Toques na tela geram corações na cor temática exata da página onde você estiver! 

## 🛠️ Stack Tecnológica

| Categoria | Tecnologia |
| :--- | :--- |
| **Framework** | Vanilla JS |
| **Linguagem** | HTML5, CSS3+, JS (ES6+) e SVG |
| **Tipografia** | Google Fonts |
| **Design** | Glassmorphism, Mesh Gradients & Keyframes |
| **Performance** | DOM Cleanup (Gestão de Memória) |
| **Infraestrutura** | 100% Amor ❤️ |

## 📂 Estrutura Arquitetural

```text
Portal-do-afeto/
│
├── index.html               # Página matriz, ponto de acesso para todos os portais.
├── script.js                # Controladores globais.
├── style.css                # Interface global (reset, fontes, glassmorphism e background).
│
├── Coracoes/                # Módulo visual interativo: chuva de amor contínua.
├── Encontro/                # Responsável por guardar a linha do tempo.
├── Galeria-de-Elogios/      # Componente rotacional com falas carinhosas.
├── Mil-te-amos/             # Listagem de mil te amos.
└── Pedido/                  # A Página mestre do pedido especial.
```
